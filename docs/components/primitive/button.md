# UButton

## Properties

### text

* type: String

### tooltip

* type: String

### variant

* type: String (success/info/warning/error/secondary/primary/danger)
* default: "info"

### disabled

* type: boolean
* default: false

### visible

* type: boolean
* default: true

### loading

* type: boolean
* default: false

### icon
* type: String
* default: ""


## Events

### click

## Examples

```html
<u-button variant="danger" disabled="false" autofocus text="Submit" visible="true" @click="" loading="true" />
```
